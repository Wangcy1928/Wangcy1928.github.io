## 赛后补题

### F题
**题意**：
对于一个 0 到 \(n-1\) 的排列 \(P\)，定义它的权值为：

\[
F(P) = \sum_{0 \le i < j < n} (P_j - P_i)
\]

给定一个从 0 到 \(n-1\) 的排列 \(P\)，以及两个整数 \(k\) 和 \(x\)（\(0 < k, x < n\)），构造排列 \(P'\)，满足以下两个条件：

- \(P'[k] = x\)（即 \(P'\) 的第 \(k\) 个元素为 \(x\)，下标从 0 开始）；
- \(F(P') \equiv F(P) \pmod{n}\)。

**输入**：  
第一行包含三个整数 \(n\)（\(1 < n < 2e5\)）、\(k\) 和 \(x\)（\(0 < k, x < n\)）。  
第二行包含 \(n\) 个整数 \(P[0], P[1], \dots, P[n-1]\)，表示给定的 0 到 \(n-1\) 的排列。

**输出**：  
如果无解，输出一行包含一个整数 \(-1\)。  
否则输出一行包含 \(n\) 个整数，表示符合条件的排列 \(P'\)。

## 思路

设 \(P'\) 是 \(P\) 循环左移 1 位后的排列，即：

\[
P'_i = P_{(i+1) \bmod n}
\]

权值公式可化简为：

\[
F(P) = \sum_{i=0}^{n-1} P_i \cdot (2i - n + 1)
\]

则循环左移 1 位后的权值差为：

\[
F(P') - F(P) = n \cdot (2P_0 - n + 1)
\]

这意味着，无论循环左（右）移多少位，排列的权值模 \(n\) 始终保持不变。

**代码**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k, x;
    cin >> n >> k >> x;
    vector<int> P(n);
    int pos = -1;
    for (int i = 0; i < n; i++) {
        cin >> P[i];
        if (P[i] == x) pos = i;
    }
    int shift = (pos - k + n) % n;
    vector<int> ans(n);
    for (int i = 0; i < n; i++) {
        ans[i] = P[(i + shift) % n];
    }
    for (int i = 0; i < n; i++) {
        cout << ans[i] << " \n"[i == n - 1];
    }
    return 0;
}
```
